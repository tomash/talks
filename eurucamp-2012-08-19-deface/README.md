# Viewing

This is a [ShowOff](http://github.com/schacon/showoff) presentation.

To see it:

    $ gem install showoff
    $ git clone git://github.com/tomash/showoff-deface-eurucamp.git
    $ cd showoff-deface-eurucamp
    $ showoff serve

And then point your browser to http://localhost:9090
